# 🚆 Train ARP Finder

A simple web app to find the **Advance Reservation Period (ARP)** opening date for IRCTC train ticket booking.

## What is ARP?

IRCTC allows booking train tickets 60 days before the journey date. This tool helps you find the exact date when booking opens for your journey.

## Technologies Used

- HTML
- CSS
- JavaScript

## How to use

1. Open the website
2. Select your journey date
3. Click **Find Reservation Period**
4. See when booking opens!
